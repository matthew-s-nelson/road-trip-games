# Road Trip Games

A mobile-friendly web app for playing classic road trip games with friends and family in the car. No internet required after loading — everything runs locally in the browser.

## Games

| Game | Players | Type |
|------|---------|------|
| [License Plate Game](docs/games/license-plate.md) | 1+ | Tracking / Collection |
| [Alphabet Game](docs/games/alphabet-game.md) | 1+ | Tracking / Racing |
| [I Spy](docs/games/i-spy.md) | 2+ | Guessing |
| [Road Trip Bingo](docs/games/road-trip-bingo.md) | 1–6 | Spotting / Bingo |

## Goals

- Simple, low-friction UI optimized for use in a moving vehicle (large tap targets, readable in sunlight)
- No account or login required
- State persists across accidental page refreshes
- Works offline after first load (PWA)
